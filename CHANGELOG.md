# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2024-12-03

### Changed
- 重构为纯 Docker 部署方式
- 简化文档，所有内容整合到 README

- 移除 systemd、scripts 等非 Docker 相关文件

- GitHub Actions 自动构建多架构 Docker 镜像
- Docker Compose 配置
- 完整的 Docker 部署文档

### Added
